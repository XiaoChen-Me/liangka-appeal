## 为什么有这个指引

首先，“两卡”处罚本意是阻止帮助网络信息犯罪，减少网络诈骗，使社会更加和谐。

但在实践中，很多人无辜遭受“两卡”处罚，这些**冤假错**的处罚，使人备受煎熬，严重影响工作与生活。

这些**冤假错**罚不仅后果极其恶劣，而且处理相当繁琐，想解除处罚，可能面临不了解流程、花钱疏通关系、占用工作时间等众多问题。

因此，为了方便快捷的让无辜受害者们解决问题，作此指引教程。

### 后果严重

我可以对两卡处罚置之不理吗？

恐怕不行，两卡处罚的直接影响：

* 暂停银行账户非柜面交易
* 禁止开立银行账户
* 银行账户余额冻结不可收付

这些影响，将导致被冤假错罚的公民不可正常收支工资薪酬、急用余额时转出失败、用工单位指定银行办卡失败致失业等严重后果。

换种说法就是：被冤假错罚的公民，将会一定程度上被剥夺经济、金融、乃至生存权利。

### 冤枉，天大的冤枉

容易被冤假错罚的情况有很多，以下是常见的几种：

* 程序员做国际外快，换人民币时收到赃款
* 学生宝妈闲鱼店、金银首饰手机实体店，交易时收到赃款
* 员工收取劳务报酬，但用工单位的支付账户收到赃款
* 申请贷款，委托美化银行卡流水，收到赃款
* 外贸小商家换汇，收到赃款
* U商投资或搬砖数字货币，收到赃款
* 被仇人、间谍机构等蓄意陷害，收到莫名赃款
* 其他，详见 [案例清单]()

以上这些情况，大部分当事人是合规合法的，一小部分是不好说的。但不论如何，对他们实施“两卡”处罚是错误的。因为他们收到赃款这档子事，是无辜的，是被骗的。

这些情况被处罚的，都属于**误判**。因为不论是从法律法规上讨论还是从司法实践上讨论，都未满足认定为“两卡”违法的构成要件。

说人话就是：“两卡”处罚是有前提的，以上没满足前提条件就处罚了，这种**冤假错**罚不仅违法且缺德。

### 我错在何处？

***！理论上***，机关对当事人实施“两卡”**处罚，必须满足以下任一条件**：

* 当事人将本人**银行卡**、手机卡实物，**出租、出借、出售**
* 当事人组织或参与电信网络新型犯罪，或者主观上对网络诈骗行为提供帮助
* 当事人将网上银行、手机**银行授权给他人登录用于转账**，且为其提供**短信验证码或人脸识别**等辅助验证
* 当事人**主观明知或客观应知**自己的所作所为涉嫌“**洗钱**”

> * 以上情况不属于误判，所以不需要申诉，以上情况的嫌疑人本指引不提供帮助。
> * 若是以上情况，以早日自首为盼，作伪证有百害而无一利。

理论上，**必须有过硬的证据**可证明以上任一条件才可以实施“两卡”处罚，**否则不可以处罚**。

***！实际上***，据我观察，机关处罚当事人只看一个条件：

* 你的银行卡收到的款项，被认定为赃款

这种一刀切的处罚，不合理、不合规、违反两卡政策的本意和初心。

## “两卡”是什么

简单的说，官方定义的两卡，主要指的是涉及电信网络诈骗活动的银行卡、手机卡。

而在本文中，“两卡”泛指一切 银行卡|手机卡|支付app 相关处罚措施。

### 两卡具体指哪些卡

手机卡：

1. 联通、移动、电信三大运营商的电话卡
2. 虚拟运营商的电话卡
3. 物联网卡

**银行卡：**

1. 个人借记卡、贷记卡、准贷记卡（一般指**储蓄卡**、信用卡）
2. 对公账户及结算卡（一般指**企业银行账户**）
3. 非银行支付机构账户（一般指**微信支付宝**等支付app）

### 什么情况属于两卡违法犯罪

“两卡”犯罪是指非法出租、出售、买卖“两卡”的违法犯罪活动。

也就是说，只要没有出租出售出借“两卡”，就不应认定为两卡违法犯罪。

但实际上，司法机关对于两卡违法犯罪的认定，误判误罚特别常见。

### 两卡处罚措施有哪些

本文中的两卡处罚措施，泛指所有 银行卡|手机卡|支付app 相关处罚措施。

这些处罚措施的处罚原因，通常是当事人（你）的银行卡收到过脏钱。赃钱是来自被诈骗受害人的。

接下来，本指引将会详细介绍两卡处罚措施的类型有哪些。

#### 司法

司法指紧急**止付**、司法**冻结**。后文统称为**司法**。

司法是**公检法机关**对单张银行卡的处罚措施、风控措施。

通常是银行卡收到赃款，并且付款人报警后，公安机关对收款人此张银行卡执行止付、冻结措施，使银行卡余额不可转出。

~~此外，也有法律诉讼对方当事人申请财产保全，法院执行冻结的情况。~~

司法后，这张银行卡将会**只收不付**，即无法转出余额、无法取现。可以收到来自其他人的转账。

司法一般是全额冻结，如果是部分冻结，超出冻结部分的余额是可以正常收付使用的。

司法是有明确期限的，一般只要没有案件，**3-7日内会自动解除**

> * 司法的处罚机关，**一般是**派出所、下辖各支队大队、反诈中心、省市公安厅局、专案组等**公安机关**
> * 如果司法期限是半年或以上，意味着你遭遇的不仅是司法，还有其他两卡处罚措施，详见后文各管控名单
> * 冻结半年及以上期限的，一般需要退赃退赔才可以解除（详见 半年以上司法解冻指南）

#### 银行相关风控

银行相关风控指限制**非柜**面交易、**不收不付**、**禁止开立账户**、降低转账额度、封禁个人线上转账功能、暂停银联电子交易等。

银行相关风控是**开户支行**、各级人民银行、开户行的本省总行等对高风险客户的**内部风控**措施。

通常是客户收到过赃款，开户行被司法机关通报批评。银行为了防止客户再次收到赃款，将客户列入**高风险客户**，执行风控措施。

或者是**受其他两卡处罚措施关联**，开户行依据其他机关下发的**管控名单**，统一执行风控措施。

> * 如果是因为其他管控名单关联风控，需要先解除其他管控名单。如涉案管控、总对总管控、断卡惩戒等。
