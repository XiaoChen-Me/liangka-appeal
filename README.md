## 为什么有这个指引

首先，“两卡”处罚本意是阻止帮助网络信息犯罪，减少网络诈骗，使社会更加和谐。

但在实践中，很多人无辜遭受“两卡”处罚，这些**冤假错**的处罚，使人备受煎熬，严重影响工作与生活。

这些**冤假错**罚不仅后果极其恶劣，而且处理相当繁琐，想解除处罚，可能面临不了解流程、花钱疏通关系、占用工作时间等众多问题。

因此，为了方便快捷的让无辜受害者们解决问题，作此指引教程。

### 后果严重

我可以对两卡处罚置之不理吗？

恐怕不行，两卡处罚的直接影响：

* 暂停银行账户非柜面交易
* 禁止开立银行账户
* 银行账户余额冻结不可收付

这些影响，将导致被冤假错罚的公民不可正常收支工资薪酬、急用余额时转出失败、用工单位指定银行办卡失败致失业等严重后果。

换种说法就是：被冤假错罚的公民，将会一定程度上被剥夺经济、金融、乃至生存权利。

### 天大的冤枉

容易被冤假错罚的情况有很多，以下是常见的几种：

* 程序员做国际外快，换人民币时收到赃款
* 学生宝妈闲鱼店、金银首饰手机实体店，交易时收到赃款
* 员工收取劳务报酬，但用工单位的支付账户收到赃款
* 申请贷款，委托美化银行卡流水，收到赃款
* 外贸小商家换汇，收到赃款
* U商投资或搬砖数字货币，收到赃款
* 被仇人、间谍机构等蓄意陷害，收到莫名赃款
* 其他，详见 [案例清单]()

以上这些情况，大部分当事人是合规合法的，一小部分是不好说的。但不论如何，对他们实施“两卡”处罚是错误的。因为他们收到赃款这档子事，是无辜的，是被骗的。

这些情况被处罚的，都属于**误判**。因为不论是从法律法规上讨论还是从司法实践上讨论，都未满足认定为“两卡”违法的构成要件。

说人话就是：“两卡”处罚是有前提的，以上没满足前提条件就处罚了，这种**冤假错**罚不仅违法且缺德。

### 我错在何处

***！理论上***，机关对当事人实施“两卡”**处罚，必须满足以下任一条件**：

（违法构成要件）

* 当事人将本人**银行卡**、手机卡实物，**出租、出借、出售**
* 当事人组织或参与电信网络新型犯罪，或者主观上对网络诈骗行为提供帮助
* 当事人将网上银行、手机**银行授权给他人登录用于转账**，且为其提供**短信验证码或人脸识别**等辅助验证
* 当事人**主观明知或客观应知**自己的所作所为涉嫌“**洗钱**”

> * 以上情况不属于误判，不冤枉、所以不需要申诉，以上情况的人员本指引不提供帮助。
> * 若是以上情况请不要参考本指引，以早日自首为盼，作伪证有百害而无一利。

**不属于以上情况而被处罚的，都是被冤枉的。**

因为，**必须有过硬的证据**可证明以上任一条件才可以实施“两卡”处罚，**否则不可以处罚**。

***！实际上***，据我观察，机关处罚当事人只看一个条件：

* 你的银行卡收到的款项，被认定为赃款

这种一刀切的处罚，不合理、不合规、违反两卡政策的本意和初心。

# “两卡”是什么

简单的说，官方定义的两卡，主要指的是涉及电信网络诈骗活动的银行卡、手机卡。

而在本文中，“两卡”泛指一切 银行卡|手机卡|支付app 相关处罚措施。

## 两卡具体指哪些卡

（官方定义）

手机卡：

1. 联通、移动、电信三大运营商的电话卡
2. 虚拟运营商的电话卡
3. 物联网卡

**银行卡：**

1. 个人借记卡、贷记卡、准贷记卡（一般指**储蓄卡**、信用卡）
2. 对公账户及结算卡（一般指**企业银行账户**）
3. 非银行支付机构账户（一般指**微信支付宝**等支付app）

## 什么情况属于两卡违法犯罪

“两卡”犯罪是指非法出租、出售、买卖“两卡”的违法犯罪活动。

也就是说，只要没有出租出售出借“两卡”，就不应认定为两卡违法犯罪。

但实际上，司法机关对于两卡违法犯罪的认定，误判误罚特别常见。

> * 为什么说是误判误罚？认定两卡犯罪的标准是什么？详见：[我错在何处](#我错在何处)

## 两卡处罚措施介绍

本文中的两卡处罚措施，泛指所有 银行卡|手机卡|支付app 相关处罚措施。

这些处罚措施的处罚原因，通常是当事人（你）的银行卡收到过脏钱。赃钱是来自被诈骗受害人的。

两卡处罚措施主要有：司法、银行风控、各类管控名单、运营商风控、支付app风控等。

### 司法

本文的司法专指紧急**止付**、司法**冻结**。后文统称为**司法**。

司法是**公检法机关**对单张银行卡的处罚措施、风控措施。

通常是银行卡收到赃款，并且付款人报警后，公安机关对收款人此张银行卡执行止付、冻结措施，使银行卡余额不可转出。

司法后，这张银行卡将会**只收不付**，即无法转出余额、无法取现。可以收到来自其他人的转账。

司法一般是全额冻结，如果是部分冻结，超出冻结部分的余额是可以正常收付使用的。

司法是有明确期限的，一般只要没有案件，**3-7日内会自动解除**

> * 司法的处罚机关，**一般是**派出所、下辖各支队大队、反诈中心、省市公安厅局、专案组等**公安机关**
> * 如果司法期限是半年或以上，意味着你遭遇的不仅是司法，还有其他两卡处罚措施，详见后文各管控名单
> * 冻结半年及以上期限的，一般需要退赃退赔才可以解除（详见 半年以上司法解冻指南）
> * 广义上说，司法并不仅限于银行卡。一切有嫌疑的资产如京东e卡、股票期货基金资产甚至房产等都可以司法冻结。

### 银行风控

银行风控指限制**非柜**面交易、**不收不付**、**禁止开立账户**、降低转账额度、封禁个人线上转账功能、暂停银联电子交易等。

银行相关风控是**开户行**、各级人民银行、开户行的本省总行等对高风险客户的**内部风控**措施。

通常是客户收到过赃款，开户行被司法机关通报批评。银行为了防止客户再次收到赃款，将客户列入**高风险客户**，执行风控措施。

或者是**受其他两卡处罚措施关联**，依据其他机关下发的**管控名单**，银行统一执行风控措施。

如果自身没有管控名单、银行卡也没有冻结，可以与开户行协商解除银行风控。一般都会给解除。

> * 欲解除银行风控，必须先解除其他管控名单。如涉案管控、总对总管控、断卡惩戒等。

### 各类管控名单

通常指各级机关下发的管控名单。管控名单是较严重的处罚措施。

常见的如：涉案管控名单、总对总管控名单、断卡惩戒名单等。

通常是银行卡收到赃款，受害人报警后，卡主未能及时处理，致使赃款被认定为一级涉案，然后被公安机关录入管控名单。

被列入管控名单，几乎所有银行都会风控。具体影响有：

* 所有银行都**禁止办新卡**
* 几乎不可能贷款，包括房车贷
* 名下所有银行卡状态会在一个月内陆续**非柜**，停止线上功能
* 其他银行风控措施（详见 [各类银行风控简表](#各类银行风控简表)）
* 运营商风控（详见后文）
* 微信支付宝使用受限（通常关联断卡惩戒名单）

各类管控名单多数没有明确的期限，即使有期限，也不建议躺平等待自动解除。

> * 涉案管控名单的期限情况如下：
>   * 期限根据各省政策不同各有差异，大部分是五年期限，个别省未明确、大概率是永久。
>   * 五年期限的，虽然理论上是期满自然解除，但是实际上很难自然解除。
>   * 仔细想一下，即便五年到期了，你没有把涉案款退给受害人，各机关会专门为你下发解控函吗；不排除你的作案嫌疑的话，各银行会冒风险把你移出行内高风险人员名单吗？很明显，都很难。
>   * 所以还是建议不要躺平，主动去走退赔、解冻、排除嫌疑的流程，其实所谓的五年期限和永久期限没什么区别。
> * 各名单只是原则上禁止办新卡，但断卡惩戒是绝对禁止办新卡。（管控内办卡指南）
> * 断卡惩戒建议主动申诉解除。到期解控的话依然禁止办新卡
> * 2023年4月前被断卡惩戒的人员，不计入个人征信。
> * 部分省份（如河北省、福建省等），涉案名单会导致运营商风控。所有手机卡封停（见后文）。

### 通信运营商风控

通信运营商风控指四大通信运营商对客户实施的各类风控措施。

通常受涉案名单、断卡惩戒名单关联影响。

风控手段可以是：

* 注销、停用手机卡
* 停用主动拨号功能，只能来电接听
* 停用短信功能
* 注销名下四运营商全部手机卡（后文统称**断手机卡**）

这些风控手段都没有具体期限，想解除风控需要先解除相关联的各类管控名单。

> * 关联断卡惩戒名单，大概率会被断手机卡
> * 部分省份涉案名单会断手机卡，如河北省、福建省等
> * 断手机卡通常会只能留一个手机号用作生活，但也可能不留

### 微信支付宝风控

微信支付宝风控指各支付app的限制措施。

通常受断卡惩戒名单、司法冻结、涉案名单关联影响。

常见的限制措施有：

* 限制收款功能，禁止收款
* 转出限额
* 停用零钱通、余额宝、支付APP内的基金账户理财账户贵金属账户
* 余额冻结，禁止转出

> * 风控不仅限于微信支付宝，也包含美团、抖音、数字人民币等一切有支付消费功能的app
> * 各app禁止支付，通常是受断卡惩戒关联。
> * 支付宝禁止收款、转出功能，有可能是涉案款项直入支付宝余额导致的。

# 怎样解除两卡

下文将尽量简明扼要的指导解除两卡处罚，主要指的是解除涉案管控名单。

因为大多数两卡措施，都是关联涉案名单的，只要解了涉案名单，其他的就都好解了。

解除两卡可能涉及的地点有户籍地、涉案卡开户行所在属地、你本人当前所在地、冻结地等。

下文一切指引你解除两卡处罚的步骤，只要提及“户籍地”相关机关的步骤，你都可以尝试替换为“涉案卡开户行所在属地”相关机关重做一次试试。

**再强调一下，下文各步中、所有提及“户籍地某某某”的步骤，都可以替换为“涉案卡开户行所在属地某某某”重试。**

## 第一步 查询管控名单

第一步，先要搞清楚自己是因为什么管控名单，导致的自己无法开卡、所有卡非柜。

尤其是：涉案人员、总对总、断卡惩戒这三个管控。

> 参考：[各类管控名单简表](#各类管控名单简表)

搞清这一步的目的是，根据解除难度判断自己有没有必要去解除。

下面是查询自己管控名单的方法，每一小步都要做：

0. **善于言辞，礼貌和智慧并存**。既然你是冤枉的，那就应该扮演好被冤枉的角色，对公安机关通常应该是窦娥冤跪求帮忙的态度，对银行机关须应变，相机决定是采取较强硬的"凭啥针对我"的态度还是同样采取求帮忙的态度。聪慧一点，切勿过分据理力争。
1. **投石问路，使用app简单测试**。如果不确定自己有没有被“全面非柜+禁止开卡”，可以试试微信支付宝能否充值，**试试民生银行app能否申领电子卡**，如果民生不允许申领电子卡，那八成是有各类管控名单的。
2. **痛过分手，查找通知你涉案的短信**。如果收到来自银行发的包含诸如“公安机关认定为涉案/人员/账户”此类的短信，就意味着有涉案管控名单。涉案短信好像比分手短信痛苦的多。
3. **迷雾有路，致电户籍地机关**。翻到短信此步可略。如果没有翻到短信也不要迷茫，致电户籍地或开卡地反诈电话96110，问为什么自己不能开卡，是什么管控，是哪个银行卡涉案，要不厌其烦的多次沟通交流。注意：公安部门内的人对这方面不一定专业，通常会念错，你要结合上文《各类管控名单简表》自己甄别。打不通是常态，多尝试几次，实在打不通就考虑亲自去一趟反诈机关当面问。
4. **大胆的上，质问涉案卡开户行**。此时你应当知道涉案卡是哪个了，动身去一趟涉案卡的开户行。你需要从柜台确认自己是受什么影响导致无法开卡，银行通常不会直接告诉你，你需要根据话术自行判断是什么名单，结合上文《各类管控名单简表》（例如问是谁下发的通知不让开卡，省市司法机关还是人民银行？）**。然后让银行把冻结机关的名称、电话给你。在纸上和微信上各牢记一份。**
5. **乘风破浪，致电冻结地机关**（可选）。银行可能会把你当嫌疑人而不理你或者搪塞你，或者银行说的话你没分析出来是啥名单。此时可以考虑拨打异地冻结机关的电话，银行没给你电话的话，就试着找户籍地或开卡属地的反诈部门要异地冻结机关的联系方式（说个合理的理由，比如争取受害人谅解、或者想退赔涉案款什么的），然后恳求冻结地帮你查询灰黑名单。冻结地通常不会拒绝，因为你退赔他们会立功，但本地机关反而很难会理你。当然，如果你认识的有本地的熟人，直接问熟人给你查天网灰黑名单就好了。这步主要是查总对总和惩戒名单而不是涉案名单，因为外省有时候会查不到涉案名单。
6. **我是影帝，装小白去开户、对照验证管控名单**。此时你应当大概已经知道自己有哪些名单了，你需要挑几个五大银行的支行网点的柜台尝试开卡，然后参照柜员告知的不开卡原因，与你前几步获得的信息相互验证。但一定要扮演被冤枉的角色，而且要挑五大行的支行网点，勿去地方行支行、各银行省分行网点，因为这些非一线、小银行柜员更容易报警，咱虽然不怕被报警，但多一事不如少一事。
7. **确保万全，查惩戒告知书和人行征信报告**。以上各步依次完成后，最终还需要验证一下自己有没有断卡惩戒名单。如果有让你签过惩戒告知书，就是被惩戒了，没有的话八成没被惩戒，可以携身份证去你本人当前所在地的市级人民银行征信业务中心打印完整版征信报告，检查报告内有没有惩戒字样，如果没有就未被惩戒。虽然通常来说断卡惩戒时司法机关会给你签发惩戒告知书，但也有不签发惩戒书、然后偷偷惩戒的。而且查询断卡惩戒通常不必去找公安等司法机关，他们查的不一定对。
8. **确保十万全，联系人行查惩戒**。如果依然不确定自己有没有被断卡惩戒，或者你是2023年以前被管控的，那么可以尝试致电户籍地市级人民银行客服热线、或中国人民银行总行官网留言查询自己是否有断卡联合惩戒的情况，因为少数情况征信报告里不体现惩戒。（也可以找公安机关查，但他们说的不一定准，你需要自己甄别有效信息）

做完这些，你大概已经明白自己有哪些名单了。

各管控解除难度及代价：

* 涉案管控单重管控-相对简单，退赔后60-80%概率能解除
* 涉案名单+总对总管控等（不含惩戒）多重管控-相对较难，退赔后50-70%概率能解除
* 三地及以上涉案、或涉案款极黑（如冒充公检法、骗银行预留验证码划扣等）-难，退赔后30-60%概率能解除
* 有断卡惩戒名单-极难，退赔后解除概率在10%以下
* 解除时可能会面临不重的行政处罚，如罚款2000、行政拘留5日诸类。
* 解除管控你或许需要陆续耗费大约共7-15日时间用于开卡属地、户籍地、冻结地多处跑道办事。
* 即使不一定需要跑道，你仍应当根据开卡属地、户籍地、目前所在地、冻结地之间的距离，初步计划通勤预算。
* 顺利的话，退赔后60日左右可解除各管控名单，正常开卡、用卡、贷款。管控及涉案情况越简单解决起来越快。

要结合你的涉案金额、涉案地有几处、是否惩戒、涉案款性质、时间成本经济成本等，自己判断有没有解除管控的必要。

这里建议你无论如何都先联系一下户籍地、冻结地机关再做决定。

因为这事主要掌握在他们几个警官手里，他们的态度很关键。

概率只是相对的，仅用于参考，说不定你会遇见热心肠有耐心的警官，动动手指就轻松解决了。

## 第二步 退赔解冻、情况说明

当你要决定解决两卡管控，首要的前提是退赔。

这一步的目的是证明自己是冤枉的。退赔、解冻之后才能证明自己是冤枉的，有解除各管控的机会。

注意：

* 本步骤的退赔指的是退回涉案款项到受害人账户，如果退赔时对方账户不是受害人，请小心。
* 本步骤的解冻指的是解除司法冻结。其他的如非柜、不收不付等与本步骤没有关系。（[什么是司法冻结](#司法)）
* 退赔解冻是没有先后顺序的，即使你原本已经解冻了，仍然需要退赔。
* 下文指引中，尝试联系户籍地机关无效的，可以试试联系涉案卡开户行所在属地机关，说不定有奇效。
* 《情况说明》很重要（见后文），一定要设法拿到。即便要不到原件，也得要到复印件或者户籍地机关手里。

下面是退赔解冻的流程与方法，每一小步都要做：

1. **知己知彼，获知基本信息**。在与各地机关沟通之后，你应当已经知道了自己涉案款项的金额是多少、受害人姓名和账户。如果不知道自己涉案款金额多少，就尝试联系冻结地、户籍地公安机关，获知涉案款金额。
2. **不预则废，做好纸质文案**。提前做好文字和语言工作。设想并整理一下将来与公安机关对话时怎样讲述自己的冤情，你所讲述的内容将可能作为笔录保存。以下是对话时应当注意的几个事项：
   * 最好是**只讲述与涉案款项单笔交易相关的情况**，包括其中的前因后果。与涉案款项无关的如其他往来款项可以不用说。说了其他往来，让你退赔涉案无关款项你退不退呢？所以你没必要讲述其余无关的款项往来。
   * 与机关进行问答时须**当心陷阱问题**，主要是你出借银行卡的目的是什么、你为什么违法犯罪、你把诈骗的钱存在哪了、你卖卡获利多少钱、为什么干这种明知违法犯罪的事等。着重思考一下这类问题你应当如何反应回答。
   * 既然你是冤屈的，那你就最好采取谦逊有礼、楚楚可人的言辞与态度。当然，这并非一成不变的，交流是一门艺术，请根据对方性格随机应变。
   * 语言风格需要**简短有力**，少一点感性，多一点理性。少一点情绪，**多一点逻辑**。太多人会踩这个坑，你如果实在不知道该怎么说，就扔给ai让他帮你整理语序与逻辑吧。
   * 你需要根据实际情况，按需强调自己没有出借银行卡、强调没有获利、强调做兼职的前因后果（如何被骗）等。
3. **证据确凿，整理好你被冤枉的证据资料**。你第2步中整理的文案，需要搭配上相关的资料用以佐证，包括聊天记录截图、交易订单截图、往来流水等，然后保存成电子版、复印件各一份。同样的，你只需整理与涉案款项相关及周边的资料，无关的资料没必要打印。
4. **敢说敢做，联系冻结地并要求出具说明**。把整理好的文案牢记在心，然后联系冻结地机关，提出你要解除司法冻结的诉求。然后冻结地机关大概会要求你退赔涉案款给受害人，此时你不妨先答应下来，然后把你整理的文案告诉冻结地机关，请求冻结地机关为你出具《不涉案证明》或《**情况说明**》。联系冻结地机关时需要注意以下几个事项：
   * 《不涉案证明》比《情况说明》更佳，《不涉案证明》部分地区也称作《排除涉案证明》。大部分机关是不愿意出具《不涉案证明》的，那么就请求冻结地机关出具《情况说明》，说明里**最好能包含“排除xxx嫌疑”字样。但切忌过分强求**，如果冻结地不是很了解这些，你**可以找个相似的模板发给冻结地**，让他们参考一下。
   * 提出退赔解冻诉求后，异地机关可能会提出诸多要求，例如让你提供各类资料材料，或者让你亲自动身去一趟异地当面办理业务等。你需要尽力遵守异地机关的要求，并按照异地机关的指引去办理。如果需要亲自去异地，你需要提前请假预留出大约7天的时间，用以应对突发情况（如行政拘留等）。如果实在不方便过去，就试探一下冻结地机关能否把案件发协查到户籍地或你当前所在地办理，如果冻结地拒绝了，那最好还是按他要求老老实实去一趟冻结地机关比较好。
   * 不论是《不涉案证明》还是《排除涉案证明》还是《情况说明》，后文都统称为《情况说明》或《说明》。
   * **尽量把**《说明》**要在自己手里**，但冻结地可能不情愿把《说明》交给个人，那就尝试要求冻结地通过“公对公”的方式把《说明》发给你户籍地的公安机关（结合下面第6步）。
5. **脚踏实地，务必走完退赔解冻流程**，不论冻结地是否同意出具《说明》。如果冻结地死也不指引你退赔，那么你**可以考虑私下退回涉案款**给受害人的账户（尽量还是听冻结地的指引，私自退赔容易使冻结地觉得你不老实），注意退回的金额要分毫不差，而且必须使用你本人的银行账户转账。
6. **死皮赖脸，继续索求说明**。冻结地的事差不多已经都做完了，以上各步骤都做完后，如果《说明》依然没有要到手里，你就去找你户籍地或开卡属地负责管辖你的警官，把你之前的冤情、整理的文案和证据资料、以及在冻结地遇到的难题、和你已经退赔的情况一股脑告诉他，乞求户籍地警官帮你联系冻结地机关索要《说明》，最好是能给你个人手里，这样后面的事处理起来更方便。
7. **说话，既是技术，也是艺术**。最后，说个小技巧。在与公安机关沟通时，虽然态度温和、可怜一点比较好，但措辞上最好使用“xxx地需要xx怎么办”、“xx情况是不是应该xx这样啊”这种方式，而不是“大哥，可不可以xxx”、“能不能帮我做xxxx”、“求求你xxx吧”，防止蹬鼻子上脸。当然，这并非一成不变的，交流是一门艺术，请根据对方性格随机应变。

经过一系列的努力，功夫不负有心人，你退赔解冻，并要到了《情况说明》。

但成功率不是100%，根据各省的政策不同、警官的性格不同，要不到《情况说明》也不罕见。

如果实在是要不到《情况说明》，此时你可以采取的方法有这些：

* 不厌其烦的电话联系并请求冻结地机关出具说明，并乞求户籍地机关帮你联系冻结地索取说明。频率以一周1次为佳。
* 前往涉案卡开户行，去柜台自称已经解决涉案款项的问题了（如果条件不允许去开户行，也可以电话联系），然后让柜员联系公安机关要资料。频率以每月1-2次为佳。
* 亲自动身去一趟冻结地机关，面述冤情索取《说明》。总共可以去1-2次，再一再二不再三。第2次去时语气可稍微强硬一点，注意分寸。
* （如果不是必要，尽量不要这样做）公安部官网留言，把你的冤情前因后果和退赔情况、冻结地不给开情况说明的事归纳整理，挂到公安部网站上。需要较强的文字表达能力、简洁干练。原则上不要提本地机关，只说异地。并且尽量不要提警官个人，要说机关。公安部留言须谨慎，假如冻结地脾气好则可以解决问题，否则留言就容易是个不吉的事。留言频率以不超过每季度1-2次为佳。
* （如果不是十分必要，请不要这样做）冻结地市级人民政府12345官网留言，和公安部留言相似，但留言总共最好不要超过2次。此方法可能会带来不幸。
* （如果不到万不得已，不要这样做）户籍地上访，强力要求户籍地向冻结地机关索要情况说明。这是个双刃剑，如果上访后未能解决问题，反而可能导致你遭受莫名诅咒，难得平安。
* 当然，如果你的头上有神灵保护，上面这些妖魔鬼怪不会近你的身，大胆留言投诉即可。但不建议普通人这样做。

如果退赔后，连续两个月没解除司法冻结，此时你可以采取的方法有这些：

* 和上面的相似，你可以致电冻结地机关催、前往涉案卡开户行让柜员帮你催、公安部/冻结地市政府留言等。
* 和上面的不同，你不可以上访、也不建议专门去一趟异地（没必要）。但可以线上留言冻结地市长热线12345敦促解冻。致电有可能搪塞你甚至不理你。

## 第三步 申诉解除各管控名单

从现在开始，冻结地就与你无关了，接下来就是户籍地、涉案卡开户行所在属地的事了。

不论查询到了多少管控名单，都优先解除涉案管控名单。因为解除涉案管控是相对比较容易的，而且解除涉案时会下发《解控函》，再继续解除其他比较困难的管控名单时《情况说明》+《解控函》可作为重要参考文件，所以会方便很多。

**注意**：

1. **异地解冻** ：首先，确保您的银行卡已经在冻结地成功退赔解冻。没有退赔解冻是很难申诉解除各管控名单的。
2. **时间间隔** ：异地解冻后，可能短时间内不会发现有任何管控名单，但后续一个多月后可能会有总对总管控。你可以间歇性的查询名单，通常半年内没有任何管控，就没事了。

申诉管控名单具体步骤：

1. **联系开户行所在地反诈机关** ：在发现有总对总管控后，第一时间联系您开户行所在地或户籍地的反诈机构，寻求反诈警官帮助并提交申诉。把各类证据材料和《说明》交给警官。
2. **提交《涉诈风险账户审查表》** ：前往涉案卡的开户行，要求他们为您办理《涉诈风险账户审查表》的流程。确保填写信息准确无误，以免延误处理。
3. **准备相关材料** ：提交申请后，开户行所在地反诈机构可能会要求您提供进一步的材料。确保准备好这些资料，以增加申诉通过的概率。
4. **接受询问** ：开卡属地、户籍地反诈机构在收到您的材料后，可能会联系您进行第二次的笔录询问。请如实回答，保持配合态度。
5. **审核与上报** ：如果反诈机构同意解除管控，他们将逐级上报至省公安厅或国家公安部。审核通过后，您的银行卡风控措施将被解除。
6. **消除银行涉案记录** ：一旦您的开户行收到盖章完毕的审查表，请及时联系他们，要求上报人行消除您的涉案记录（即涉案黑名单）。这样，您在其他银行开卡或使用身份证时，将不会显示涉案信息。
7. **等待审核结束**：大概需要等待1-2个月，然后就可以正常开卡、用卡了。此时你应密切跟进审核进度，找哪个机关申诉的就联系谁，跟进频率以每半个月1-2次为佳。

常见问题

* **沉默是罪** ：在整个过程中，务必保持与开户行和反诈机构的沟通，了解进度和要求。适当的间歇性联系申诉机关查询审核进度。
* 材料和智商同理，不能残缺 ：确保提交的所有材料都是完整且真实的，避免因资料不全导致申诉失败。尤其是《情况说明》。
* **耐心等待一俩月** ：解除管控和消除涉案记录的过程可能需要一定时间，请保持耐心，及时跟进进展。
